Hello! Welcome to the V-HR. Web application built by Pavle Paunovic

Guide:

  1. Enter Name And Surname and press button Add employee to add new employee
  2. You can search employee in input bellow List of Employees
  3. Click on the employee name to open the form
  4. Enter the data in the form
  5. To remove employee click on X next to employee in list

Note: This modal will show only first time loading the web application
